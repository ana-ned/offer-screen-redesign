# Uncapped — Offer Page Prototype

An interactive prototype of Uncapped's redesigned Offer page — one screen where the offer *is* the calculator (no separate calculator, no second draw page).

## Products

Switchable in the demo panel:

- **Refinance & Top-Up**
- **Term Loan**
- **Cash Advance**
- **Line of Credit**

Each shows a read-only summary on the left that recalculates live as you adjust the controls on the right.

## How to use

- Open the deployed link (or `index.html` in any browser).
- Press **`D`** — or click the **"Demo controls"** tab (top-right) — to switch products and toggle offer elements on/off.
- Every configuration is saved to the URL, so you can **copy the link to share an exact setup**.

## Notes

- Numbers are illustrative.
- Single self-contained HTML file — no build step, no dependencies.
